## Featured Project: Supply Chain Digital Twin

**Repository:** `sit-2026-digital-twin`
*An Enterprise-Grade Analytical Engine built with Python, Streamlit, and DuckDB.*

This Master's thesis project evaluates the financial and operational feasibility of nearshoring within European maritime transport. Rather than presenting a static supply chain model, this repository houses a simulation engine designed to predict exact impacts on cost, transit times, and carbon emissions prior to physical implementation or capital investment.
